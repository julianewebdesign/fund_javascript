Variáveis, tipos de dados.

Variáveis: Variáveis são contêineres que armazenam dados que podem ser manipulados durante a execução do código.

Tipos de Dados: JavaScript possui vários tipos de dados primitivos:

String: Sequência de caracteres.

Number: Números, sejam inteiros ou de ponto flutuante.

Variáveis: Armazenam dados que podem ser usados e manipulados.

Boolean: Representa valores verdadeiros ou falsos.

Null: Representa a ausência intencional de valor.

Undefined: Valor padrão de uma variável que não foi inicializada.

Object: Coleções de pares chave-valor.

Estruturas de Controle: Essas estruturas determinam o fluxo do programa com base em condições ou repetições.
